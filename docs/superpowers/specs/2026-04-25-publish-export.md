## What This Builds

A dedicated export page at `/listings/[id]/publish` that lets the user copy per-platform listing fields, paste the resulting listing URL back, and transition the listing to `published` status.

**Done when:** Can copy all eBay fields, paste the listing URL, and listing status transitions to `published`.

## API Route

`PATCH /api/listings/[id]/publish`

Request body (all optional, at least one required):
```json
{ "platform": "ebay" | "poshmark", "listing_url": "https://..." }
{ "mark_published": true }
```

Behavior:
- `listing_url` + `platform`: read current `listing_urls` JSONB, merge `{ [platform]: url }`, write back
- `mark_published: true`: set `status = 'published'`
- Both can be in the same request
- Returns `{ ok: true, status, listing_urls }` or `{ error: string }`

Uses service role client (same pattern as `upload/route.ts` — inline `createClient` from `@supabase/supabase-js`).

## SEO Audit Checks

Derived from listing data, no external calls:

| Check | Pass condition |
|-------|---------------|
| eBay title set | `platform_fields.ebay.title` exists |
| eBay title ≤ 80 chars | `title.length <= 80` |
| eBay title ≥ 3 words | word count ≥ 3 |
| eBay description set | `ebay.description` exists and ≥ 50 chars |
| eBay item specifics | `Object.keys(item_specifics).length >= 3` |
| Poshmark title set | `platform_fields.poshmark.title` exists |
| Poshmark title ≤ 60 chars | `title.length <= 60` |
| Poshmark description set | `poshmark.description` exists and ≥ 50 chars |
| Price set | `suggested_price_cents != null` |

## Platform Field Layout

**eBay:** title → category_id → condition_id → description → each item_specific key/value. Copy All produces a labeled text block.

**Poshmark:** title → description → category → size → original_price (if set). Copy All produces a labeled text block.

---

## Status Transition

"Mark as Published" button appears when at least one platform URL has been saved. Clicking sends `{ mark_published: true }` to the API. On success, a green "Published" badge replaces the current status display and the button is disabled.
